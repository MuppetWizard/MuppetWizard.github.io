修饰符

Java 和 Kotlin 函数可见性修饰符对照表
| 修饰符 | Java | Kotlin |
| :—: | :—: | :—: |
| public | 所有类可见 | 所有类可见（默认）|
| private | 当前类可见 | 当前类可见 |
| protected | 当前类、子类、同一包路径下的类可见 | 当前类、子类可见 |
| default | 同一包路径下的类可见（默认） | 无 |
| internal | 无 | 同一模块中的类可见 |
